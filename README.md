# Amigo Secreto

Aplicação web simples para sortear aleatóriamente seus amigos no jogo de fim de ano Amigo Secreto

## Funções

- Adciona nomes de amigos à lista
- Validação de entrada do usuário
- Lista de Amigos 
- Sorteio

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
